# Hotel Transilvania - Sistema de Gestao Hoteleira

## Visao Geral

O **Hotel Transilvania** e um sistema completo de gestao hoteleira desenvolvido em Java com Spring Boot.
O sistema oferece funcionalidades para gerenciamento de reservas, hospedagens, servicos e relatorios.

### Funcionalidades Principais

- Gestao de Quartos e Tipos de Quarto
- Reservas Online
- Check-in/Check-out
- Controle de Servicos Adicionais
- Relatorios em PDF com JasperReports
- Interface Web Responsiva

### Tecnologias Utilizadas

- **Backend**: Java 21, Spring Boot 3.x
- **Banco de Dados**: SQL Server
- **Relatorios**: JasperReports
- **Build**: Maven
- **Frontend**: Thymeleaf, Bootstrap
